# Dashboard

The dashboard is the start page after login and provides a quick overview of your own vacation status as well as -- for admins -- the status of the entire team.

---

## Employee View

As an employee, you see three information areas on the dashboard:

### Remaining Vacation

Shows the **remaining vacation days** for the current year. This number takes into account:

- Annual entitlement
- Carryover from the previous year
- Already taken (approved) vacation days
- Bonus days and adjustments
- Expired days

### Open Requests

Shows the number of your own vacation requests with the status **Pending** (in progress). A click leads directly to the list of your own vacations.

### Next Vacation

Shows the next planned (approved) vacation with start and end date. If no future vacation is planned, a corresponding message is displayed.

---

## Admin View

Admins see all information from the employee view plus the following:

### Pending Approval Requests

Shows the number of open vacation requests from **all employees** that are waiting for approval or rejection. A click leads directly to the pending requests page.

!!! tip "Tip"
    Check pending requests regularly so that employees receive timely feedback on their vacation requests.

### Team Status

Shows which employees are **absent today** or **during the current week**. This area gives admins a quick overview of current team capacity.

---

## Navigation from the Dashboard

The following areas can be reached directly from the dashboard:

- **Calendar** -- via the side navigation
- **My Vacations** -- via the link at open requests or the side navigation
- **Vacation Requests** (admin only) -- via the link at pending approval requests
