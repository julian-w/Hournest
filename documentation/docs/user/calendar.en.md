# Calendar

The calendar shows a monthly view with all relevant information about holidays, weekends, and vacations.

---

## Monthly View

The calendar displays one month at a time in a grid view (weekdays as columns, weeks as rows). Each day shows the day number and -- if applicable -- vacation and holiday information.

### Navigation

- **Left arrow** -- switch to the previous month
- **Right arrow** -- switch to the next month
- **Today button** -- jump back to the current month and highlight today's date

---

## Color Coding

The days in the calendar are color-coded:

| Color / Marking     | Meaning                          |
|---------------------|----------------------------------|
| Gray background     | Weekend (Saturday, Sunday)       |
| Holiday marking     | Public holiday                   |
| Yellow / Amber      | Vacation with status **Pending** (awaiting approval) |
| Green               | Vacation with status **Approved** |
| Red                 | Vacation with status **Rejected** |

!!! info "Holidays and Vacation Calculation"
    Holidays are not counted as vacation days. If a vacation spans a holiday, that day is automatically deducted.

---

## Visibility

### Employee

Employees see **only their own vacations** in the calendar. Holidays and weekends are visible to everyone.

### Admin (Administrator)

Admins see the **vacations of all employees** in the calendar. Each vacation entry shows the employee's name, making it easy to see who is absent when.

!!! note "Planned: Group Visibility"
    Group-based visibility is planned for Phase 2. This will allow employees within a group to also see the vacations of their colleagues.

---

## Legend

At the bottom of the calendar, there is a legend explaining the color coding:

- **Pending** -- Vacation request in progress (awaiting approval)
- **Approved** -- Vacation request approved
- **Rejected** -- Vacation request rejected
- **Holiday** -- Public holiday
- **Weekend** -- Weekend (Saturday/Sunday)

---

## Usage Tips

- Use the calendar to check whether holidays fall within your desired vacation period before submitting a request
- Admins can use the calendar to monitor team absences and identify overlaps
- The calendar updates automatically when vacation requests are approved or rejected
